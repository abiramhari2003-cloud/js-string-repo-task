# JavaScript Utilities Repository

This repository contains simple JavaScript scripts demonstrating common string, array, and variable manipulations. These scripts are often useful for beginners looking to practice basic programming concepts such as:

- String normalization and transformation
- Generating usernames and checking password strength
- Formatting data for backend APIs
- Basic data processing and utility functions

## Files

- `array.js` - Example operations and exercises with arrays.
- `intro.js` - Introduction script (likely contains basic syntax examples).
- `string.js` - A collection of string utilities including user authentication helpers, backend formatting, and data processing simulations.
- `variable.js` - Demonstrates usage and manipulation of variables in JavaScript.

## Usage

Each file can be run with Node.js:

```bash
node string.js
node array.js
node intro.js
node variable.js
```

Feel free to modify and extend these examples as you learn more about JavaScript.
